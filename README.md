PETCONNECT: A FULL-STACK APPLICATION FOR ANIMAL ADOPTIONS AND SERVICES

We can find the already deployed site here: https://petconnect-production-85dc.up.railway.app/

LINK TO THE GITHUB REPOSITORY: https://github.com/LacramioaraDan/PetConnect.git

APPLICATION BUILD STEPS:

To build the application, we run: "npm run build"

APPLICATION INSTALLATION AND LAUNCH STEPS:

For the INSTALLATION part, we need to clone the repository and install npm using the following commands:

"git clone <(https://github.com/LacramioaraDan/PetConnect.git)>" 
    -> 
"cd folder_name"
    -> 
"npm install"

And for LAUNCHING the application, we use the following commands:

For the web page: "npm run dev"

For the node environment: "npm run dev-node"
